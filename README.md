# 🎬 Netflix Movie Data Analysis

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a Netflix movie dataset to uncover insights about movie trends, genres, ratings, and popularity.  
The analysis helps understand **what type of content performs best on Netflix** and how movie trends have evolved over time.

---

## 🎯 Objectives
- Analyze movie genres distribution
- Identify most popular movies
- Study rating trends
- Analyze release year trends
- Find most common genres
- Discover popularity patterns

---

## 📁 Dataset Information

**Dataset Name:** Netflix Movie Dataset  
**File Name:** `mymoviedb.csv`

### Dataset Features

| Column Name | Description |
|-------------|-------------|
| Title | Movie Name |
| Genre | Movie Genre |
| Release_Date | Movie Release Date |
| Popularity | Popularity Score |
| Vote_Average | Average Rating |
| Vote_Count | Number of Votes |
| Original_Language | Movie Language |
| Overview | Movie Description |
| Poster_Url | Movie Poster URL |
